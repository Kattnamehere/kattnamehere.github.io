First Website

Welcome to my first website and my first big project! I'm excited to share my thoughts, experiences, and insights with you all through this blog. It's been a journey getting to this point, and I can't wait to see where it leads. I made this website in just over a week, which is pretty good considering I'm not the best designer. This was all created using HTML, CSS, and JS only, no fancy dynamic web app libraries here! It's all hosted on GitHub, so if you're interested, you can browse through all of my commits and see the progress from the beginning. If you're wondering about the font, it's called Mint Grotesk.

Blog

You might not see much from the git commits because I was mostly working on a local copy of the Git repository and not actively committing to another branch, but this blog page has been in the works for a while now, right around when I added the "blog" button to the header.

Mobile Optimizations

I'm really proud to say that this website is fully optimized for mobile (at least well enough for most mobile devices). Here are some of the techniques I used:

  • Media Queries: Adjusting the layout for different screen sizes.
  • Flexible Grids: Implementing CSS Grid and Flexbox for adaptable layouts.
  • Viewport Meta Tag: Ensuring proper scaling on mobile devices.
If you encounter any issues with the mobile experience, please let me know, and I'll try to fix it as soon as possible!

Resources and Tools

Here are some of the resources and tools that were helpful during the development of this website:

  • MDN Web Docs: Comprehensive documentation on HTML, CSS, and JavaScript.
  • GitHub: For version control and hosting the website.
  • WebStorm: My IDE of choice for coding with advanced features and integrated tools.
  • Lighthouse: An open-source, automated tool for improving the quality of web pages, including performance, accessibility, and SEO.
  • W3C CSS Validator: A service provided by W3C to check the validity of your CSS code and ensure it meets the CSS standards.
  • WAVE: A web accessibility evaluation tool that helps identify and fix accessibility issues on your website.
